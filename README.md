# 🏛️ Projet-Infrastructure-Moderne-IT
**Infrastructure Système & Réseau Complète - 2026**

Ce dépôt contient le dossier technique de référence (**89 pages**) documentant la conception et le déploiement d'une infrastructure d'entreprise sécurisée de A à Z. 

Ce projet est le résultat d'un travail intensif sur la **reproductibilité** et la **rigueur technique**.

### 🛠️ Périmètre Technique :
* **Services d'Identité :** Déploiement complet de Windows Server 2022 & Active Directory (AD DS, DNS, DHCP).
* **Hardening & GPO :** Mise en place de stratégies de groupe avancées pour le durcissement de la sécurité des postes et serveurs.
* **Gestion de fichiers :** Configuration de serveurs de fichiers sécurisés avec gestion fine des droits d'accès.
* **Supervision :** Monitoring de l'état de santé de l'infrastructure via Zabbix.
* **Industrialisation du Déploiement :** Mise en place d'un serveur **WDS & MDT** pour le déploiement automatisé d'images OS (Lite Touch) sur les différents VLANs.
